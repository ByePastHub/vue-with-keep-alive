## vue-with-keep-alive

### 介绍

>`vue-with-keep-alive`可以帮助你实现前进刷新，返回时之前的页面还处于激活的状态，可以像`app`一样的体验。哪里需要缓存，把`router-view`组件使用`keep-router-view`来替换。

#### 安装

vue2.x
```
npm install vue-with-keep-alive@2
```

vue3.x
```
npm install vue-with-keep-alive
```
